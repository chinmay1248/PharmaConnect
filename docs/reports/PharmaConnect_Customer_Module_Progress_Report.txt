PHARMACONNECT CUSTOMER MODULE
Short Progress Report

Date: 8 April 2026

1. What PharmaConnect Is

PharmaConnect is a medicine supply chain platform connecting:
- Customer
- Retailer
- Wholesaler
- Company

The customer app is for:
- searching medicines,
- comparing nearby pharmacies,
- uploading prescriptions,
- placing orders,
- making payments,
- and tracking delivery.

3. Initial Project State

At the beginning, the project had only text documents and image files.
There was no mobile app, no frontend code, and no backend code.


4. What Was Done

A new Expo React Native TypeScript mobile app was created in:
- mobile/

Customer frontend prototype was built with:
- splash screen
- onboarding screen
- mock login screen
- home tab
- search tab
- orders tab
- profile tab
- medicine detail view
- nearby retailer comparison view
- cart view
- order tracking view

Backend and integration work now added:
- Express backend foundation in `backend/`
- Prisma schema and migration
- seed data for medicines, salts, diseases, retailer inventory, and sample users
- medicine search API
- medicine detail API
- retailer comparison API
- mobile discovery flow connected to live backend data with mock fallback
- mobile signup/login/profile/address flow connected to backend auth routes
- customer order creation connected to backend order APIs with local fallback
- prescription upload metadata, payment confirmation, tracking, invoice, and order history connected to backend APIs
- customer notification inbox connected to backend notification APIs
