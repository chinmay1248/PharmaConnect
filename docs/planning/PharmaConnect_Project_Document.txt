1. Project Overview
PharmaConnect is a full-stack pharmaceutical supply chain platform that digitally connects every stakeholder in the medicine industry — the end customer, the retail pharmacy, the wholesale distributor, and the manufacturing company.

Built as a mobile-first product for Android and iOS using React Native, with web dashboards for business users, PharmaConnect is designed specifically for the Indian pharmaceutical market.

2. Tech Stack
Every technology was chosen for three reasons: it uses JavaScript (one language across the entire stack), it has the best free tier for an early-stage Indian startup, and it can scale to millions of users without changing the architecture.

Authentication
Phone number + OTP via Firebase Auth
Google Sign-In via Firebase Auth

Medicine Discovery
Search by medicine name, salt composition, or barcode scan
All medicines with same active salt shown — searched brand appears first
Medicine detail: composition, diseases cured, dosage, pack size, MRP
Substitute medicines shown: same salt, different brand, price comparison
Rx required badge and Generic badge per medicine

Filters & Sorting

Ordering
Select retailer from nearest-first sorted list with stock, price, distance, rating
Add to cart — one retailer per order (MVP)
Prescription upload via camera or gallery for Rx medicines
Real-time prescription approval/denial status from retailer
Home delivery or store pickup selection
Payment: Razorpay (UPI, cards, wallets), Cash on Delivery
Real-time order tracking with push notifications at every step
GST invoice with PDF download
Order history, saved addresses, feedback and rating


Selling Side — Managing Customer Orders
Real-time order notifications from customers
View order details including uploaded prescription image
Approve or deny orders — denial sends reason to customer
Pack, dispatch, and confirm delivery workflow
Auto-inventory deduction on dispatch using FIFO batch logic
Generate and send GST invoice to customer

Search medicines to restock
View frequent wholesalers and all available wholesalers
Compare schemes and prices across wholesalers

Auto-inventory update when stock is received

Buying Side — Sourcing from Companies
Request company to establish a supplier relationship
Place bulk orders to approved companies

PharmaConnect uses PostgreSQL with Prisma ORM. All 33 tables are grouped into 8 logical groups. Every foreign key has an explicit index. All polymorphic relationships are resolved with typed nullable columns and CHECK constraints.
